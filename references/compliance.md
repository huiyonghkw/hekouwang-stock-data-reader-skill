# 合规护栏（出任何报告前必读 · 一票否决）

这套 Skill 服务的账号**不持金融牌照**，只做「基于公开数据的中立事后复盘 + 行业科普」。下面是硬红线，命中任意一条 → 停下改写，不要产出。

## 绝不做（全平台通用）
1. **不荐股、不预测涨跌、不给买卖点/目标价/评级**。禁词：买入、卖出、抄底、加仓、减仓、止损、目标价、看多/看空、翻倍、稳赚、年化收益、带你上车、进群领票。
2. **不暗示自己是持牌/投顾主体**，昵称/简介/正文/图都不行。
3. 不诱导跨境投资、外汇、虚拟货币；不倒卖研报。

## 必做
- 每份报告**文末固定风险提示条**：
  `本文为基于公开数据的事后复盘与行业科普，不构成任何投资建议，不预测股价涨跌。市场有风险，决策需独立判断。`
- 标注数据来源与截止时点（akshare / 公司年报季报 / 截至 YYYY-MM-DD）。
- 营销号数字常错，一律以 akshare 财报/乐咕口径二次核实后再用。

## 渠道分流（关键）
- 个股报告**主发公众号 / 头条**；**小红书亦可发，但只限"复盘 / 算账 / 名词科普"这一形态**。实测「利通电子"10转4"算账」在 AI 主账号发布通过审核、1507 展现跑赢同批常规选题（2026-07 复盘），此前"绝不上小红书"的判断被证伪。
- 上小红书的**硬前提（缺一不可）**：中立事后复盘、不荐股 / 不预测 / 不给买卖点、文末风险提示条、**零站外导流**（不留公众号 / 微信 / 外链 / 「回X我发你」）、标题走**解惑式 / 算账 / 名词对照**（"这笔钱到底发了多少""XX 名词一张图看懂"）。
- **别上小红书的高风险版**：带明确方向判断 / 目标价 / 翻倍·上车话术，或**以"主力 / 龙虎榜席位 / 资金流进出"为主钩子**的"机构观感"版——这类在小红书判荐股、易高风险下架。这类仍只走公众号 / 头条。

## 措辞基调
事后复盘、描述"发生了什么"、解释机制；用"数据显示/截至 X 日"这类客观句式。把判断留给读者，不替读者决策。
